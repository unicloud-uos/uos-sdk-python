# 安装

本文介绍如何安装Python SDK。

## 环境要求

-   Python版本：2.7以上。
-   有紫光云账户，并开通OSS服务。

## 下载SDK

下载地址：[unicloud-oss-sdk-python](ttps://portal.oss-cn-north-1.unicloudsrv.com/impact/unicloud-oss-sdk-python.rar)。

- SDK基于AWS boto3编写，需要安装boto3库。

    安装步骤：

    ```
    pip install boto3
    ```

- Github地址：[点击查看](https://github.com/liyu0420/oss-python-sdk)
